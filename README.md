# Finance Operations Performance Dashboard

## Overview
An Excel-based finance dashboard developed to monitor budget performance, departmental expenses, and financial KPIs for management reporting.

## Business Problem
Finance teams often rely on multiple spreadsheets, making it difficult to monitor spending and financial performance efficiently.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas

## Key KPIs
- Budget vs Actual
- Total Expenses
- Approval Rate
- Transaction Count

## Outcome
Developed an interactive dashboard that provides a centralized view of finance operations and supports better decision-making.
